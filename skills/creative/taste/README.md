# Taste (Creative base skill)

Anti-slop frontend design skill: reads a brief, infers the right design
direction, and ships landing pages / portfolios / redesigns that do not look
templated.

## Provenance

Vendored from the upstream skill installable via:

```bash
npx skills add Leonxlnx/taste-skill
```

Source: [`Leonxlnx/taste-skill`](https://github.com/Leonxlnx/taste-skill)
(specifically `skills/taste-skill/SKILL.md`).

It is bundled here under `skills/creative/` so it ships as part of the
**Creative base skill set** and is seeded into `~/.spark/skills/` by
`src/tools/skills_sync.py` on every install — no network access required, so
it works offline. To refresh against upstream, re-fetch `SKILL.md` from the
repo above and re-apply the Spark `metadata.spark` frontmatter block.
